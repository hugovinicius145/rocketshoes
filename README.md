# RocketShoes
## Projeto resultante do Bootcamp 8.0

### Projeto Criado em React.
### Requisitos:
<ul>
  <li>Nodejs</li>
  <li>Yarn</li>
</ul>


### Executar Aplicação WEB:

#### Clonar o Repositório

#### Entrar na Pasta

#### No terminal escrever:

*Baixa os Arquivos Necessários.
>  yarn

* Inicie a API Fake
> yarn json-server server.json -p 3333

Em outro terminal!
*Executa o Projeto.
>  yarn start
